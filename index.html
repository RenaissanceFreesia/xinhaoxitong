🔁 卷积与LTI系统

卷积是LTI系统时域分析的核心，输出等于输入与冲激响应的卷积。

\[y(t) = x(t) * h(t) = \int_{-\infty}^{\infty} x(\tau)h(t-\tau)d\tau\]
  • 交换律：x*h = h*x
  • 结合律：(x*h1)*h2 = x*(h1*h2)
  • 分配律：x*(h1+h2) = x*h1 + x*h2

📡 傅里叶变换（频域分析）

将信号从时域转换到频域，揭示频率成分。

\[X(j\omega) = \int_{-\infty}^{\infty} x(t)e^{-j\omega t}dt\]
  • 时移性质：x(t−t0) ↔ X(jω)e−jωt0
  • 卷积定理：时域卷积 ↔ 频域相乘
  • 信号带宽、滤波、调制核心工具

📶 奈奎斯特抽样与信号恢复

抽样频率 fs > 2fmax 时，可无失真恢复原信号。

\[f_s > 2f_{max}\]
  • 抽样：x[n] = x(nT)
  • 零阶保持、sinc函数插值恢复
  • fs 过低会产生混叠，无法还原信号
