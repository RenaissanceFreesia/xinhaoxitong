📌 1. 基本信号定义

正弦信号：x(t)=A sin(ωt+φ)，周期、频率、振幅、相位四要素

余弦信号：x(t)=A cos(ωt+φ)，与正弦相位差 90°

方波：只有高低两种电平，富含奇次谐波，常用于数字电路

锯齿波：线性上升+快速回落，含所有整数次谐波

📌 2. 频率与周期关系

周期 T：完成一次振动时间；频率 f=1/T；角频率 ω=2πf

例：f=1Hz → T=1s → ω=2π rad/s

📌 3. 振幅与相位

振幅 A：信号最大偏离量，决定强弱

相位 φ：决定 t=0 时刻位置，影响信号同步

📌 4. 信号分类

连续时间信号：t 连续（如正弦波）

离散时间信号：t 只在整数点有定义

周期信号：满足 x(t+T)=x(t)，无始无终

非周期信号：不重复，如单个脉冲

📌 5. 线性时不变系统（LTI）

满足线性（叠加+齐次）和时不变（延迟不变形）

核心：输入 → 系统 → 输出；可用卷积描述

📌 6. 傅里叶级数（FS）

任何周期信号可分解为：直流+基波+各次谐波正弦/余弦之和

意义：时域 ↔ 频域，揭示信号频率组成
